# computer-vision-opencv
# Image Processing & Object Detection with OpenCV

This project explores various Computer Vision techniques using Python and the OpenCV library. It was developed and tested in a Google Colab environment.

## Key Functionalities
* **Edge Detection:** Implementing Canny and Sobel filters to identify object boundaries.
* **Image Transformation:** Grayscale conversion, blurring (Gaussian), and thresholding.
* **Morphological Operations:** Using erosion and dilation for noise reduction.
* **Data Visualization:** Using Matplotlib to display processing stages.

## Tools & Libraries
* **Python 3**
* **OpenCV:** Core library for computer vision tasks.
* **NumPy:** Efficient array manipulation for image data.
* **Google Colab:** Cloud-based development environment.

## How to Run
You can open the `.ipynb` file directly in Google Colab or Jupyter Notebook to view the code and the rendered outputs.
